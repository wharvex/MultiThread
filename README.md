# MultiThread

## Prerequisites

Download `ojdbc17.jar` from the [Oracle JDBC Downloads page](https://www.oracle.com/database/technologies/appdev/jdbc-downloads.html) and place it in the `lib/` directory.

## How to run

**Compile:**
```
javac -cp "lib/*.jar" -d out OracleScheduledQuery.java
```

**Run:**
```
java -cp "out;lib/*.jar" OracleScheduledQuery
```

# MultiThread

## Prerequisites

Download `ojdbc8.jar` from the [Oracle JDBC Downloads page](https://www.oracle.com/database/technologies/appdev/jdbc-downloads.html) and place it in the `lib/` directory.

## How to run

**Compile:**
```
javac -cp lib\ojdbc8.jar -d out OracleScheduledQuery.java
```

**Run:**
```
java -cp out;lib\ojdbc8.jar OracleScheduledQuery
```
